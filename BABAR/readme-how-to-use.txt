Com a aplicação aberta,
se deseja criar outra tabela, clique em criar tabela.
digite o nome dela e aperte enter esta tabela será criada com os atributos:
(ID, nome, valor_compra, valor_venda, imagem).

Para adicionar produtos, escreva no menu principal qual tabela está sendo usada.
digite o ID o nome e os valores de compra e venda e selecione uma imagem ícone deste.

Para procurar produtos, escreva no menu principal qual a tabela está sendo usada
clique em Buscar e selecione o ID que deseja.
Atenção!! No momento da busca, o ícone é baixado do banco de dados e salvo na pasta
"C:\\Users\\Public\\Documents\\" !!
